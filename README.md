# PromptForge

Светлый/тёмный минималистичный **конструктор промтов** для ИИ. Пошагово: категория → тип → настройки → готовый промт (RU/EN). Плюс библиотека готовых промтов.

**Возможности**
- 8 категорий, 38 типов (код, изображения, текст, видео, аудио, ассистент, бизнес, обучение)
- Двуязычные заготовки (RU/EN) и уровень детализации (Кратко/Норма/Максимум)
- Живой предпросмотр промта на шаге настроек
- Библиотека из 35 готовых промтов с поиском
- История, шаблоны, экспорт `.txt`/`.md`, шаринг ссылкой
- Кнопки «Открыть в ChatGPT / Claude / Gemini»
- Глобальный поиск, тёмная тема, PWA (офлайн)

**Стек:** чистый HTML/CSS/JS, без сборки.

## Запуск локально
Открыть `index.html` в браузере. Для PWA/офлайна и корректных ссылок — отдавать по http:
```
python -m http.server 5173
```
и открыть `http://localhost:5173`.
